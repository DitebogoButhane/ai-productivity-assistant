# AI Workspace Productivity Assistant

## Project Overview

The **AI Workspace Productivity Assistant** is a modern, responsive web application designed to help professionals automate everyday workplace tasks using Artificial Intelligence (AI). The application improves productivity by summarizing meeting notes, organizing daily tasks, and generating structured work plans.

This project focuses on providing an intuitive user experience without requiring user registration, login, or database storage. Users can immediately access the application's AI-powered features through a clean and responsive interface.

---

## Features Implemented

### 📝 Meeting Notes Summarizer
- Convert lengthy meeting notes into concise summaries.
- Extract key discussion points.
- Identify decisions made during meetings.
- Highlight action items, deadlines, and responsibilities.

### 📅 AI Task Planner / Scheduler
- Generate structured daily task plans.
- Prioritize tasks based on urgency and importance.
- Organize workloads for improved productivity.
- Present tasks in an easy-to-follow schedule.

### 🎨 Modern User Interface
- Responsive design for desktop, tablet, and mobile devices.
- Clean and intuitive navigation.
- Fast-loading interface with minimal user interaction.

### ⚡ No Authentication Required
- No user registration.
- No login functionality.
- No database required.
- Instant access to all application features.

## Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/ai-workspace-productivity-assistant.git
```

### 2. Navigate to the Project Directory

```bash
cd ai-workspace-productivity-assistant
```

### 3. Install Dependencies (if applicable)

```bash
npm install
```

### 4. Configure Environment Variables

Create a `.env` file in the project root and add your API key:

```env
OPENAI_API_KEY=your_api_key_here
```

> If you are using Lovable's built-in AI integration, this step may not be required.

### 5. Run the Application

For development:

```bash
npm run dev
```

Or simply open the project if using the Lovable platform.
